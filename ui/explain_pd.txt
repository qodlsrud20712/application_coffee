1. code, name에 값을 입력 후
   추가 버튼을 누르면 해당 DB와 Ui 테이블에 추가됨.

2. Ui 테이블 화면에서 변경하고 싶은 Value가 있는 행에서
   마우스 우클릭을 하면 행이 선택되면서 수정, 삭제, 검색 목록이 뜸.
   1) 수정 클릭 시 선택된 행의 값들이 컬럼명과 같은 칸에 입력됨.
      그리고 값을 수정 후 수정 버튼을 클릭하면 수정 완료.
   2) 삭제 클릭 시 선택된 행이 삭제. 삭제 버튼 클릭 시 동일하게 수행.
   3) 검색 클릭 시 선택된 행의 코드에 해당하는 제품의 이름이 출력.
